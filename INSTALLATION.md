# HOW TO SETUP

### Source Code

1. [레포](https://github.com/29CM-Developers/frontend-29cm-test-northfacegawd) 를 클론한다.

```shell
$ git clone git@github.com:29CM-Developers/frontend-29cm-test-northfacegawd.git
```

2. 디펜던시를 설치한다.

```shell
$ npm install
```

3. 로컬에 프로젝트를 실행한다.

```shell
$ npm run dev
```
